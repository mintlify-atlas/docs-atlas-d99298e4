Poketto Stack includes Docker configuration for containerized deployment.

## Dockerfile

The project includes a multi-stage Dockerfile:

```dockerfile
FROM node:20-slim AS builder
ENV PNPM_HOME="/pnpm"
ENV PATH="$PNPM_HOME:$PATH"
RUN corepack enable

WORKDIR /app
COPY . /app

RUN --mount=type=cache,id=pnpm,target=/pnpm/store pnpm install
RUN pnpm run build

FROM nginx:alpine
COPY docker/nginx.conf /etc/nginx/conf.d/default.conf
COPY --from=builder /app/dist /app/dist
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

## Building the image

Build the Docker image:

```bash
docker build -t poketto-stack .
```

With build args:

```bash
docker build \
  --build-arg NODE_ENV=production \
  -t poketto-stack:latest .
```

## Running the container

Run the container:

```bash
docker run -p 8080:80 poketto-stack
```

With environment variables:

```bash
docker run \
  -p 8080:80 \
  -e VITE_API_URL=https://api.example.com \
  poketto-stack
```

## Docker Compose

Create a `docker-compose.yml`:

```yaml
version: '3.8'

services:
  app:
    build: .
    ports:
      - "8080:80"
    environment:
      - VITE_API_URL=https://api.example.com
    restart: unless-stopped

  # Add other services (database, redis, etc.)
  postgres:
    image: postgres:16
    environment:
      POSTGRES_DB: myapp
      POSTGRES_USER: user
      POSTGRES_PASSWORD: password
    volumes:
      - postgres_data:/var/lib/postgresql/data
    ports:
      - "5432:5432"

volumes:
  postgres_data:
```

Run with Docker Compose:

```bash
docker-compose up -d
```

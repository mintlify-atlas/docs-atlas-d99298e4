## Introduction

The Tabs component allows you to organize content into multiple panels, showing only one panel at a time. Built on Radix UI's Tabs primitive, it provides accessible, keyboard-navigable tab interfaces perfect for settings pages, product views, dashboards, and multi-step forms.

## Installation

```tsx
import {
  Tabs,
  TabsList,
  TabsTrigger,
  TabsContent,
} from "@poketto/ui";
```

## Usage

Basic tabs structure with three sections:

```tsx
<Tabs defaultValue="account">
  <TabsList>
    <TabsTrigger value="account">Account</TabsTrigger>
    <TabsTrigger value="password">Password</TabsTrigger>
    <TabsTrigger value="settings">Settings</TabsTrigger>
  </TabsList>
  <TabsContent value="account">
    Account settings content
  </TabsContent>
  <TabsContent value="password">
    Password settings content
  </TabsContent>
  <TabsContent value="settings">
    General settings content
  </TabsContent>
</Tabs>
```

### Controlled Tabs

Manage tab state externally:

```tsx
function ControlledTabs() {
  const [activeTab, setActiveTab] = useState("overview");
  
  return (
    <div>
      <Tabs value={activeTab} onValueChange={setActiveTab}>
        <TabsList>
          <TabsTrigger value="overview">Overview</TabsTrigger>
          <TabsTrigger value="analytics">Analytics</TabsTrigger>
          <TabsTrigger value="reports">Reports</TabsTrigger>
        </TabsList>
        
        <TabsContent value="overview">
          Overview content
        </TabsContent>
        <TabsContent value="analytics">
          Analytics data
        </TabsContent>
        <TabsContent value="reports">
          Reports and insights
        </TabsContent>
      </Tabs>
      
      <p className="mt-4 text-sm text-muted-foreground">
        Current tab: {activeTab}
      </p>
    </div>
  );
}
```

## Props

### Tabs

The root tabs container.

<ParamField path="defaultValue" type="string">
  The value of the tab that should be active by default.
</ParamField>

<ParamField path="value" type="string">
  The controlled value of the active tab. Use with `onValueChange`.
</ParamField>

<ParamField path="onValueChange" type="(value: string) => void">
  Callback fired when the active tab changes.
</ParamField>

<ParamField path="orientation" type="'horizontal' | 'vertical'" default="horizontal">
  The orientation of the tabs.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes for the tabs container.
</ParamField>

### TabsList

Container for the tab triggers.

<ParamField path="className" type="string">
  Additional CSS classes. Default includes inline-flex layout, rounded background with muted styling.
</ParamField>

<ParamField path="children" type="ReactNode">
  TabsTrigger components.
</ParamField>

### TabsTrigger

Clickable tab button.

<ParamField path="value" type="string" required>
  Unique identifier matching the corresponding TabsContent value.
</ParamField>

<ParamField path="disabled" type="boolean">
  When true, prevents user interaction with the trigger.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes. Active state automatically applies background and shadow.
</ParamField>

<ParamField path="children" type="ReactNode">
  The label content for the tab.
</ParamField>

### TabsContent

Content panel for each tab.

<ParamField path="value" type="string" required>
  Unique identifier matching the corresponding TabsTrigger value.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes. Default includes top margin and focus ring styles.
</ParamField>

<ParamField path="children" type="ReactNode">
  Content to display when this tab is active.
</ParamField>

## Accessibility

- Built on Radix UI Tabs following WAI-ARIA tabs pattern
- Keyboard navigation: `Tab` to focus tab list, `Arrow keys` to navigate between tabs, `Space` or `Enter` to activate
- Automatic focus management when switching tabs
- Screen reader support with proper ARIA roles and attributes
- Active tab indicated visually with background color and shadow
- Focus visible ring for keyboard navigation

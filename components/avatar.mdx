The Avatar component displays user profile pictures with automatic fallback handling. Built on Radix UI's Avatar primitive, it gracefully handles loading states and missing images.

## Installation

```bash
npm install @poketto/ui
```

## Import

```tsx
import { Avatar, AvatarImage, AvatarFallback } from "@poketto/ui";
```

## Basic Usage

```tsx
<Avatar>
  <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
  <AvatarFallback>CN</AvatarFallback>
</Avatar>
```

## Component API

### Avatar

The root container for the avatar component.

**Props:** Extends `Radix UI Avatar.Root` props

- `className`: Additional CSS classes
- All Radix UI Avatar.Root props

**Default styles:**
- Size: `h-10 w-10` (40px)
- Shape: Rounded full (circular)
- Overflow: Hidden
- Flex display with shrink-0

### AvatarImage

The image element within the avatar.

**Props:** Extends `Radix UI Avatar.Image` props

- `src`: Image source URL
- `alt`: Alternative text for accessibility
- `className`: Additional CSS classes
- All Radix UI Avatar.Image props

**Default styles:**
- Aspect ratio: Square
- Size: Full width and height of container

### AvatarFallback

The fallback content shown when the image is unavailable or loading.

**Props:** Extends `Radix UI Avatar.Fallback` props

- `className`: Additional CSS classes
- `children`: Fallback content (typically initials or an icon)
- All Radix UI Avatar.Fallback props

**Default styles:**
- Background: Muted
- Layout: Centered flex
- Shape: Rounded full (circular)
- Size: Full width and height of container

### Dynamic Avatar Component

```tsx
interface UserAvatarProps {
  user: {
    name: string;
    avatar?: string;
  };
  size?: "sm" | "md" | "lg";
}

function UserAvatar({ user, size = "md" }: UserAvatarProps) {
  const sizeClasses = {
    sm: "h-8 w-8 text-xs",
    md: "h-10 w-10 text-sm",
    lg: "h-16 w-16 text-lg",
  };

  const initials = user.name
    .split(" ")
    .map((n) => n[0])
    .join("")
    .toUpperCase()
    .slice(0, 2);

  return (
    <Avatar className={sizeClasses[size]}>
      <AvatarImage src={user.avatar} alt={user.name} />
      <AvatarFallback>{initials}</AvatarFallback>
    </Avatar>
  );
}
```

## Accessibility

- Always provide an `alt` attribute for `AvatarImage`
- Use meaningful fallback text (initials are recommended)
- The component automatically handles loading and error states
- Powered by Radix UI for built-in accessibility features
- Consider adding tooltips with full names for better UX

## Integration with Radix UI

This component is built on `@radix-ui/react-avatar`, which provides:

- Automatic image loading state management
- Graceful fallback handling
- Optimized rendering performance
- Built-in accessibility features

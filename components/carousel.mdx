The Carousel component provides a flexible, accessible carousel with built-in navigation controls, keyboard support, and responsive touch interactions powered by Embla Carousel.

## Installation

```bash
npm install @poketto/ui embla-carousel-react
```

## Import

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselPrevious,
  CarouselNext,
} from "@poketto/ui";
```

## Basic Usage

```tsx
<Carousel>
  <CarouselContent>
    <CarouselItem>Slide 1</CarouselItem>
    <CarouselItem>Slide 2</CarouselItem>
    <CarouselItem>Slide 3</CarouselItem>
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```

## Component API

### Carousel

The root carousel container.

**Props:** Extends `React.HTMLAttributes<HTMLDivElement>`

- `orientation`: `"horizontal"` | `"vertical"` (default: `"horizontal"`)
- `opts`: `CarouselOptions` - Embla Carousel options
- `plugins`: `CarouselPlugin[]` - Embla Carousel plugins
- `setApi`: `(api: CarouselApi) => void` - Callback to receive the carousel API instance
- `className`: Additional CSS classes

### CarouselContent

The container for carousel items.

**Props:** Extends `React.HTMLAttributes<HTMLDivElement>`

- `className`: Additional CSS classes

### CarouselItem

Individual carousel slide.

**Props:** Extends `React.HTMLAttributes<HTMLDivElement>`

- `className`: Additional CSS classes

**Default behavior:**
- Full width slides (`basis-full`)
- Use classes like `md:basis-1/2` for responsive multi-item views

### CarouselPrevious

Previous navigation button.

**Props:** Extends Button component props

- `variant`: Button variant (default: `"outline"`)
- `size`: Button size (default: `"icon"`)
- `className`: Additional CSS classes

**Features:**
- Automatically disabled when at the start (unless loop is enabled)
- Positioned absolutely on the left (horizontal) or top (vertical)

### CarouselNext

Next navigation button.

**Props:** Extends Button component props

- `variant`: Button variant (default: `"outline"`)
- `size`: Button size (default: `"icon"`)
- `className`: Additional CSS classes

**Features:**
- Automatically disabled when at the end (unless loop is enabled)
- Positioned absolutely on the right (horizontal) or bottom (vertical)

### useCarousel

Hook to access carousel context and controls.

```tsx
const {
  api,
  scrollPrev,
  scrollNext,
  canScrollPrev,
  canScrollNext,
  orientation,
} = useCarousel();
```

## Carousel Options

Common Embla Carousel options:

```tsx
<Carousel
  opts={{
    align: "start", // "start" | "center" | "end"
    loop: true, // Enable infinite loop
    skipSnaps: false, // Skip snaps that are not visible
    dragFree: false, // Enable free scroll
    containScroll: "trimSnaps", // "trimSnaps" | "keepSnaps"
  }}
>
```

See [Embla Carousel Options](https://www.embla-carousel.com/api/options/) for complete documentation.

## Advanced Usage

### Access Carousel API

```tsx
import { type CarouselApi } from "@poketto/ui";

function MyCarousel() {
  const [api, setApi] = React.useState<CarouselApi>();
  const [current, setCurrent] = React.useState(0);
  const [count, setCount] = React.useState(0);

  React.useEffect(() => {
    if (!api) return;

    setCount(api.scrollSnapList().length);
    setCurrent(api.selectedScrollSnap() + 1);

    api.on("select", () => {
      setCurrent(api.selectedScrollSnap() + 1);
    });
  }, [api]);

  return (
    <div>
      <Carousel setApi={setApi}>
        <CarouselContent>
          <CarouselItem>Slide 1</CarouselItem>
          <CarouselItem>Slide 2</CarouselItem>
          <CarouselItem>Slide 3</CarouselItem>
        </CarouselContent>
        <CarouselPrevious />
        <CarouselNext />
      </Carousel>
      <div className="text-center mt-2">
        Slide {current} of {count}
      </div>
    </div>
  );
}
```

### Custom Navigation

```tsx
function CustomCarousel() {
  const { scrollPrev, scrollNext, canScrollPrev, canScrollNext } = useCarousel();

  return (
    <div>
      <button onClick={scrollPrev} disabled={!canScrollPrev}>
        Previous
      </button>
      <button onClick={scrollNext} disabled={!canScrollNext}>
        Next
      </button>
    </div>
  );
}
```

### Dot Indicators

```tsx
function CarouselWithDots() {
  const [api, setApi] = React.useState<CarouselApi>();
  const [current, setCurrent] = React.useState(0);

  React.useEffect(() => {
    if (!api) return;
    api.on("select", () => {
      setCurrent(api.selectedScrollSnap());
    });
  }, [api]);

  return (
    <div>
      <Carousel setApi={setApi}>
        <CarouselContent>
          {slides.map((slide, index) => (
            <CarouselItem key={index}>{slide}</CarouselItem>
          ))}
        </CarouselContent>
      </Carousel>
      <div className="flex justify-center gap-2 mt-4">
        {slides.map((_, index) => (
          <button
            key={index}
            className={`h-2 w-2 rounded-full ${
              index === current ? "bg-primary" : "bg-muted"
            }`}
            onClick={() => api?.scrollTo(index)}
          />
        ))}
      </div>
    </div>
  );
}
```

## Plugins

Embla Carousel supports plugins for extended functionality:

```bash
npm install embla-carousel-autoplay embla-carousel-fade
```

```tsx
import Autoplay from "embla-carousel-autoplay";
import Fade from "embla-carousel-fade";

<Carousel
  plugins={[
    Autoplay({ delay: 3000, stopOnInteraction: true }),
    Fade(),
  ]}
>
  {/* Content */}
</Carousel>
```

Available plugins:
- `embla-carousel-autoplay`: Auto-play functionality
- `embla-carousel-auto-scroll`: Continuous scrolling
- `embla-carousel-class-names`: Add classes to slides
- `embla-carousel-fade`: Fade transitions

See [Embla Carousel Plugins](https://www.embla-carousel.com/plugins/) for more.

## Accessibility

- Keyboard navigation with Arrow Left/Right keys
- ARIA roles: `region` with `carousel` role description
- Slides have `group` role with `slide` role description
- Navigation buttons include screen reader text
- Touch/swipe support on mobile devices
- Focus management for keyboard users

Built on `embla-carousel-react` with features:

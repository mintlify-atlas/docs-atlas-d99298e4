## Introduction

Poketto Stack includes a comprehensive component library built on top of [shadcn/ui](https://ui.shadcn.com/) and [Radix UI](https://www.radix-ui.com/) primitives. The library provides over 50 accessible, customizable components that follow modern design patterns and best practices.

All components are:
- **Accessible**: Built on Radix UI primitives with ARIA compliance
- **Customizable**: Styled with Tailwind CSS and class-variance-authority
- **Type-safe**: Full TypeScript support with proper type definitions
- **Framework-ready**: Works seamlessly with React and Next.js

## Installation

Components are exported from the `@poketto/ui` package. Install the package in your app:

```bash
pnpm add @poketto/ui
```

## Usage

Import components directly from their specific paths:

```tsx
import { Button } from '@poketto/ui/button'
import { Card, CardHeader, CardTitle, CardContent } from '@poketto/ui/card'
import { Input } from '@poketto/ui/input'

export function MyComponent() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Example</CardTitle>
      </CardHeader>
      <CardContent>
        <Input placeholder="Enter text..." />
        <Button>Submit</Button>
      </CardContent>
    </Card>
  )
}
```

## Styling

All components use Tailwind CSS for styling and can be customized through:

1. **CSS Variables**: Theme colors and design tokens
2. **className Props**: Override styles on individual instances
3. **Tailwind Config**: Extend the default theme in `tailwind.config.ts`

## Accessibility

Components follow WAI-ARIA best practices and include:
- Proper semantic HTML
- Keyboard navigation support
- Screen reader compatibility
- Focus management
- ARIA attributes and roles

---
title: Aspect Ratio
description: Maintain consistent aspect ratios for content
---

The Aspect Ratio component maintains a consistent width-to-height ratio for content, built on Radix UI Aspect Ratio primitive.

## Installation

```tsx
import { AspectRatio } from "@poketto/ui/aspect-ratio"
```

## Usage

### Basic aspect ratio

```tsx
import { AspectRatio } from "@poketto/ui/aspect-ratio"

export default function Example() {
  return (
    <div className="w-[450px]">
      <AspectRatio ratio={16 / 9}>
        <img
          src="https://images.unsplash.com/photo-1588345921523-c2dcdb7f1dcd"
          alt="Photo"
          className="rounded-md object-cover w-full h-full"
        />
      </AspectRatio>
    </div>
  )
}
```

### Different ratios

```tsx
<div className="grid gap-4">
  <div className="w-full">
    <h3 className="text-sm font-medium mb-2">16:9 (Widescreen)</h3>
    <AspectRatio ratio={16 / 9} className="bg-muted">
      <div className="flex items-center justify-center h-full">
        16:9
      </div>
    </AspectRatio>
  </div>

  <div className="w-full">
    <h3 className="text-sm font-medium mb-2">4:3 (Standard)</h3>
    <AspectRatio ratio={4 / 3} className="bg-muted">
      <div className="flex items-center justify-center h-full">
        4:3
      </div>
    </AspectRatio>
  </div>

  <div className="w-full">
    <h3 className="text-sm font-medium mb-2">1:1 (Square)</h3>
    <AspectRatio ratio={1} className="bg-muted">
      <div className="flex items-center justify-center h-full">
        1:1
      </div>
    </AspectRatio>
  </div>
</div>
```

## Props

<ParamField path="ratio" type="number" default="1">
  The desired aspect ratio (width / height)
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes
</ParamField>

## Use cases

- Video players
- Image galleries
- Card layouts
- Responsive embeds

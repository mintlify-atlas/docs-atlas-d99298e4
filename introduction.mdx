<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your development environment set up in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Poketto Stack repository to your local machine.
      
      ```bash
      git clone https://github.com/TinsFox/poketto-stack.git
      cd poketto-stack
      ```
    </Step>
    
    <Step title="Install dependencies">
      This project uses pnpm for package management. Install dependencies for all packages in the monorepo.
      
      ```bash
      pnpm install
      ```
    </Step>
    
    <Step title="Start the development server">
      Launch the admin application in development mode.
      
      ```bash
      pnpm admin:dev
      ```
      
      <Note>
        The development server will start at `http://localhost:5173` with hot module replacement enabled.
      </Note>
    </Step>
    
    <Step title="Explore the components">
      Launch Storybook to browse and interact with all available components.
      
      ```bash
      cd apps/admin
      pnpm storybook
      ```
      
      Storybook will open at `http://localhost:6006` where you can explore all UI components in isolation.
    </Step>
  </Steps>
</div>
